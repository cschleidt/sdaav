Motivation

The objective of the project is to identify a location for a new bar in New York. We want to find a "sweet spot" where there is an active party scene but in a "nice" area. The reader of this document should get a basic understanding of New Yorks areas zip codes and understand why we choose to open a bar in the chosen location.

The data

The "Party Bar" dataset indicates where the party scene is, while a combination of other attributes will guide us to the "nice" neighbourhoods. Data for the project are based on NYC OpenData and Zillow.com. NYC OpenData provides different statistics, for example registrered complaints (311). Zillow.com provides property price information.

  1. Rodent Complaints
  2. Graffiti Complaints
  3. Air quality Complaints
  4. Urinating in Public Complaints
  5. Property Market Value
  6. Number of trees
  7. Condition of trees (share of trees that are in good condition)

Visualizing the model

Zip Code Map of New York - color coded with 20 clusters including indicators for Party and Niceness

Using the values found and shown in the data exploration section we defined a combined value for the complaints, the property value and the quality of the trees. We call this value "Niceness" which is a score from 0-1 which shows how pleasant a zip code is compared to the others.

We have grouped the zip codes into 20 clusters, which have similar trademarks of complaints and property values and tree quality. We used these to show the clusters by color. the bars on the left show the niceness "score" and the party "score" for the individual zipcodes. . Using this illustration you can look into each zip code to see how it compares to the rest.

The visualization illustrates which zip codes are clustered together and the geographic position of the clusters. By hovering over the zipcodes it additionally shown the calculated values for Niceness and Party/Music

Interactive bubble chart: "Party vs. Niceness"

Shows the development of the two attributes for each of the four chosen clusters over time. We want the location of our bar to be hip and up-and-coming, so we will look for zipcodes with a positive path.

Looking at the previous illustration we have chosen the cluster we deemed to be best in regards to both niceness and party/music. These have been plotted on scatterplot, and the change over time can be shown by moving the slider or automatically by cliking the button. Each circle represents a zip code, and the size represents the combined value of both niceness and party/music

Conclusion

Looking at the previous plot, we want a zip code which is near the top right corner, and has a large circle.

We had already narrowed the search down to the 17th cluster. And by looking at the zip codes we see that 10014 stands out as it is almost as nice as the other neighborhoods but has significantly more of a party scene.

Which shows that despite there not being one location which wins on all the scores, this location is the best solution for our situation

This has led us to the area in which we could place our bar
